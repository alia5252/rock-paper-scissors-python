# Rock Paper Scissors Game (Python)

A feature-rich Rock Paper Scissors game built using Python. The game supports multiple modes, AI difficulty levels, score tracking, match history, achievements, and more.

## Features

### Game Modes
- Player vs Computer
- Player vs Player
- Tournament Mode (Best of 3, 5, or 7)

### AI Difficulty Levels
- Easy
- Medium
- Hard

### Tracking Systems
- High Score System
- Leaderboard
- Match History with Date & Time
- Statistics Tracking
- Achievement System

### Special Features
- Hidden Moves using `getpass`
- Hint Mode
- Cheat Mode
- Secret Gun Mode
- Replay System
- Player Profiles

## Technologies Used

- Python
- Random Module
- Datetime Module
- File Handling
- Exception Handling

## How to Run

### Clone the Repository

```bash
git clone https://github.com/alia5252/rock-paper-scissors-python.git
```

### Move into the Project Directory

```bash
cd rock-paper-scissors-python
```

### Run the Program

```bash
python Rock_Paper_Scissor.py
```
